Connect a free Firebase database so your scores are saved across all devices:

  1. Go to console.firebase.google.com and sign in
  2. Click Add project → give it any name → Continue
  3. In the left menu click Build → Realtime Database
  4. Click Create database → choose a region → Start in test mode → Enable
  5. Copy the URL shown at the top (e.g. https://your-app-default-rtdb.firebaseio.com)
  6. Paste it below and click Save
