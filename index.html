חברו מסד נתונים Firebase חינמי כדי שהתוצאות ישמרו בכל המכשירים:

  1. היכנסו ל-console.firebase.google.com
  2. לחצו Add project → תנו שם → Continue
  3. בתפריט השמאלי לחצו Build → Realtime Database
  4. לחצו Create database → בחרו אזור → Start in test mode → Enable
  5. העתיקו את הכתובת שמופיעה למעלה (לדוגמה: https://your-app-default-rtdb.firebaseio.com)
  6. הדביקו למטה ולחצו שמור
